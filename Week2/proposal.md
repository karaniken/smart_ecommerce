
# Smart E-Commerce – Project Proposal

**Project Title:** SmartShop – Online E-Commerce Platform  
**Student:** Kennedy Karani (BBIT/2024/56963)  
**Unit:** BIT3208 – Advanced Web Design and Development  

## Theme
E-commerce system for selling electronic gadgets and clothing.

## Core Features
- User registration and login (role-based: admin/customer)
- Product catalog with categories and search
- Shopping cart and checkout simulation
- Order management and payment simulation (M-Pesa / cash on delivery)
- Admin panel for product and order management

## Technologies
- **Backend:** PHP (procedural with PDO)
- **Database:** MariaDB (MySQL compatible)
- **Frontend:** HTML5, CSS3, JavaScript (vanilla), Bootstrap 5
- **Version control:** Git + GitHub
- **Deployment:** Localhost (XAMPP alternative: LAMP on Arch Linux)
